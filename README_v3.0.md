## 🛠 Technology Stack

### Frontend
- **HTML5**: Semantic, accessible markup
- **CSS3**: Responsive grid/flexbox, animations, glassmorphism
- **JavaScript**: Vanilla (canvas scroll, form validation, menu toggle)

### Responsive Breakpoints
- **Desktop** (1200px+): Full multi-column layouts
- **Tablet** (768-1024px): 2-3 column grids
- **Mobile** (<768px): Single column + hamburger menu

### Performance
- Minified CSS/JS ready
- Lazy-loaded images
