## 🎯 Project Overview

**SymbioSkin** is a breakthrough wearable bio-integrated device that revolutionizes thermal comfort in smart buildings through:

✨ **Individual Thermal Adaptation** - Personalized micro-climate management at body level  
🤖 **Predictive AI** - Anticipates discomfort before it happens  
🔌 **Smart Building Integration** - Real-time IoT sync with HVAC systems  
♻️ **Energy Harvesting** - Self-powered device reducing battery dependency  

### The Value Proposition
Transform the way buildings and humans interact thermally, moving from **"Macro Heating/Cooling for Everyone"** to **"Micro-Adaptive Comfort for Each Person"**.

## 🎨 Design System

### Color Palette
- **Primary**: Mint Green/Cyan (#00FF80, HSL 145°)
- **Secondary**: Tech Cyan (#00D4FF, HSL 195°)
- **Accent (Problem)**: Red (#e71d36)
- **Background**: Dark theme (#06070a) with glassmorphism

### Typography
- **Headlines**: Outfit (700-800 weight)
- **Body**: Plus Jakarta Sans (400-500 weight)
- **Code/Technical**: Monospace throughout

### Animations
- **Scroll Reveals**: 0.8s ease with cubic-bezier easing
- **Hover Effects**: 0.6s smooth transitions
- **Interactive**: 3D tilt effects, pulse animations
- **Performance**: GPU-accelerated, 60fps target
